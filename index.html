Introduction to Data Science

The course introduces students to applied examples of data collection, processing, transformation, management, and analysis to provide students with a hands-on introduction to the data science experience. Students will explore key concepts related to data science, including applied statistics, information visualization, text mining, and machine learning. “R,” the open-source statistical analysis and visualization system, will be used throughout the course. R is reckoned by many to be the most popular choice among data analysts worldwide; having knowledge and skill to use it is considered a valuable and marketable job skill for most data scientists. After taking this course, the students will be able to:

  1. Students will be able to understand:
    • a. Essential concepts and characteristics of data
    • b. Scripting/code development for data management using R and R-Studio
    • c. Principles and practices in data screening, cleaning, and linking
    • d. Communication of results to decision makers
  2. At the end of the course, students are expected to be able to:
    • a. Identify a problem and the data needed for addressing the problem
    • b. Perform basic computational scripting using R and other optional tools

Information Visualization IST 719

A broad introduction to data visualization for information professionals. Students will develop a portfolio of resources, demonstrations, recipes, and examples of various data visualization techniques. After taking this course, the students will be able to:

  1. Use R to do basic data cleaning and preparation on a wide range of datasets
    • a. Use functions to summarize and compare fields
    • b. Find missing values
    • c. Use subsets or filter data
    • d. Retype data into correct format
  2. Identify stories in datasets through exploration
    • a. Use R to create appropriate rough plots to identify distributions and relationships in the data
    • b. Use data subsetting and filtering to home in on questions of interest
    • c. Transform data through processing, linking, aggregation, summarization, and searching
    • d. Organize and manage data at various stages of a project life cycle
    • e. Determine appropriate techniques for analyzing data
  3. Create rich visual artifacts that communicate data stories
    • a. Identify the optimal type of visualization to minimize viewer cognitive overload and maximize image interpretability
    • b. Enhance viewer cognition through context cues
    • c. Use basic design principles to enhance viewer receptivity and convey meaning
    • d. Use Adobe Illustrator to combine R data visualizations, design elements, and context cues into a single artifact

Scripting for Data Analysis IST 652

Scripting for the data science pipeline. Acquiring, accessing, and transforming data in the forms of structured, semistructured, and unstructured data.
After taking this course, the students will be able to:

  1. Write scripts to access and amass data from fields in structured data, access fields in semistructured data, and define and find patterns of data in unstructured data
  2. Prepare and transform data to produce data summaries, lists, and networks
  3. Analyze and solve data access problems for the three types of data and find and deploy appropriate software packages that can be integrated into the problem solution

Quantitative Reasoning for Data Science IST 772

Classical statistical procedures used in information transfer research. Emphasis on the underlying rationale for each procedure and on criteria for selecting procedures in a given research situation.
After taking this course, the students will be able to:

  1. Demonstrate knowledge of contemporary inferential statistical concepts (from the perspective of two contemporary philosophies) and data analysis strategies by making sensible choices about:
    • a. How data collection, the data themselves, and the analysis processes relate to the kinds of inferences that can be drawn
    • b. What kinds of analysis will be feasible and developing the skill of planning data collection and measurement to facilitate appropriate analysis
  2. Practice effective data science analytics:
    • a. Preparing data for analysis, including screening data, dealing with missing data, and doing data transformations
    • b. Testing assumptions that data must meet for analyses and inferences to be reasonable
    • c. Interpreting data analysis results and outputs and communicating them to others using language that accurately describes uncertainty
  3. d. Leaving a documentation/provenance trail for other analysts to follow and reproduce your work
  4. Demonstrate competence and/or mastery of the skills needed for use of a popular statistics and data management platform to conduct sound and reproducible analyses including:
    • a. Installing R and R-studio, and creating readable code to conduct analyses
    • b. Exploring the limitations of existing data sets and how their provenance influences what analyses to perform and what inferences to draw
    • c. Choosing appropriate R procedures and configuring the relevant operational parameters

Data Administrative Concepts & Management IST 659

This is an introductory course in database management systems. It examines data structures, file organizations, concepts, and principles of database management systems (DBMS), as well as data analysis, database design, data modeling, database management, and database implementation. More specifically, it introduces hierarchical, network, and relational data models; entity-relationship modeling; basics of Structured Query Language (SQL); data normalization; and database design. This course provides hands-on experience in database design and implementation through assignments, lab exercises, and course projects. This course also introduces advanced database concepts such as transaction management and concurrency control, distributed databases, multitier client-server architectures, database applications, improving query performance through indexing, and advanced data query patterns for extract, transform, load (ETL).
After taking this course, the students will be able to:

  1. Describe fundamental data and database concepts, including various storage models.
  2. Explain and use the database development life-cycle and data models
  3. Analyze business problems and design and implement appropriate data-oriented solutions using the relational data storage model
  4. Solve problems by constructing database objects and queries using SQL.
  5. Identify performance and data integrity improvements of existing database designs and implementations
  6. Evaluate and select approaches for data migrations, temporal data, and data normalization
  7. Critique the effectiveness of DBMS in computer information systems

Advanced Big Data Management IST 769

An analysis of relational and nonrelational databases and their corresponding database management system architectures. Learn to build complex database objects to support a variety of needs from both the big data and traditional perspectives. Data systems performance, scalability, and security.
After taking this course, the students will be able to:

  1. Demonstrate comprehension of advanced issues with the relational database model such as transactions, performance, and security to understand the need for other database models
  2. Compare different database models such as document, key-value, column-family, streaming, search, graph, object, and relational
  3. Identify the most suitable database systems for a specific application’s data storage requirements
  4. Explain the CAP theorem, which identifies a system’s trade-offs of partition tolerance, availability, and consistency to describe how any given database system’s architecture fits within this context
  5. Evaluate relational, Hadoop, and NoSQL database tooling as to understand their underlying similarities and necessary differences

Introduction to Information Security IST 623

IST623 Introduction to Information Security is intended to teach fundamental elements of information security and introduce the key areas of security challenges, countermeasures, and real-life examples. The course will focus on a comprehensive understanding of information security rather than a specific security area. Topics include security properties, vulnerabilities, cryptography, security policies, access control, authentication, firewalls, wireless security, Internet security protocols, security management, security evaluation, and case studies.
After taking this course, the students will understand the comprehensive aspect of information security and be able to:

  1. Explain the fundamental security properties, challenges, countermeasures, and real-life examples
  2. Describe the key concepts, technical elements, and their tradeoffs in cryptography, access control, authentication, wireless security, firewalls, Internet security protocols
  3. Explain in-depth security knowledge and skills in the security case studies that they selected
  4. Demonstrate the hands-on ability in host protection, cryptography, secure communication, and network security analysis, using various security services and tools
  5. Exhibit the ability in research, presentation, and Q&A discussion about information security
  6. Begin to develop specialty on a specific area of information security in their further study and career preparation, extending the knowledge and skills learned from this course.
